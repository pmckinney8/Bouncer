Registration Form Example

Input Field Validation Checks

Email

  • Checks if the format is correct

First/Last Name

  • Minimum length of 2 characters
  • Only letters are allowed

Username

  • Minimum length of 6 characters
  • Maximum length of 12 characters
  • No special characters
  • Must contain atleast one number

Password

  • Minimum length of 8 characters
  • Maximum length of 15 characters
  • Must have atleast one Uppercase letter
  • Must contain atleast one number

Phone Number

  • Minimum length of 8 characters
  • Maximum length of 11 characters
  • Must not contain any letters

Credit Card

  • Checks for correct credit card input
  • fake good number = 0123765443210190

Stop Submit function (beta)

  • Won't allow the user to submit form until no errors left
